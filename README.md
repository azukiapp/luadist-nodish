# luadist-nodish

This project is a metapackage to install [nodish](https://github.com/lipp/nodish) in [luadist](luadist.org)

# Install

```bash
$ git clone https://github.com/azukiapp/luadist-nodish
$ luadist install luadist-nodish
```
